Features

The CellWatch parental control APP provides parents a set of tools to make sure their child’s technology is being used in accordance with the parents permissions. The features are optional so the parent can select which features to apply depending on their personal circumstances.

Beacon (Room & Vehicle)

Allows the parent to decide when the phone works and when it doesn’t.

Child Phone Data

Allows the parent to view the child’s call logs, text messages, browser history and location history.

Web History

Allows the parent to view the child’s surfing activity and block inappropriate web sites.

Location Tracking

See where you child’s phone has been for the past 3 days (15 min increments).

How it Work

The App is designed to provide parental control options as to how and when the child’s phone can be used. It also provides call, text and web history plus location tracking.

Note: The FREE trial does not include the home or auto beacon hardware. The home(child’s room) and auto beacon are available for purchase at checkout for $19.95 each.

Home Beacon - This beacon allows you to set a specific time (ex. 11pm – 6 am) where the child’s phone will not work except for emergency calling to either 911 or an alternate number to be set by the parent. The beacon is limited to one room – it was designed for the child’s bedroom to keep them off the phone during sleeping hours.

Auto Beacon – The beacon goes in the car and prevents a teen driver from texting, emailing or talking on the phone when the vehicle is moving.
